填写说明 / Filling Instruction
1.  “序号”一栏中填写自然数列，按照增序排列。
     “No”: Natural number, in increased order.
2.  “试验项目”一栏中填写需要进行的零件或系统试验名称。
     “Test Item”: Name of component test or system test.
3.  “试验标准/出处”一栏中填写试验所依据的标准编号； 若某些试验没有参考的标准，则填写试验所依据的其它相关资料，例如图纸等技术资料。
     “Test Spec.”: Standard No.. If there is no standards for refering, fill with other related documents, such as drawing, etc.
4.  “试验方法”一栏中概述试验所采用的方法(标准中的某条），如果没有试验标准支持，供应商应针对该试验项目制定一份详细的试验说明书。
     “Test Procedure”: Selected test method ( some paragraph in standards). If there is no standards for refering, supplier should make a detailed test spec. for the test item.
9. “试验结束”一栏中填写完成试验的时间。
5.  “限值”一栏中填写该试验项目规定试验结果的应达到的值，如果标准能够涵盖则不要填写。
     “Limit”: Reasonable value for test result.
6.  “试验单位”一栏中填写负责实际执行该试验项目的单位或者机构，包括第三方委托试验机构等。
     “Test Site”: Test situation or company execute the test.
7  "认证类型“请填写DV(Design Validation),PV(Product Validation),Others.
      “Remark”: Any other issue suppliers need to supplement.
11. 正式的工程开发验证计划，必须得到SMTC 认证工程师和设计发布工程师和高级经理的批准。
      Final ADV Plan must be approved by SMTC MVE , DRE and Senoor Manager.
8.  “试验开始”一栏中填写开始试验的时间。
       “Start Date”: Date start to test.
      “Finish Date”: Date on which the test finished.
10. “备注”一栏填写供应商需要补充说明的问题。
