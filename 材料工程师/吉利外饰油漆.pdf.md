本标准代替Q/JLY J7111122B-2019《汽车外饰油漆件技术要求》和Q/JLY J7111122A-2018《汽车
外饰油漆件技术要求》，与Q/JLY J7111122B-2019 相比，除编辑性修改外，主要技术变化如下： 
This Standard substitutes Q/JLY J7111122B-2019 “Technical Requirements for Automotive Exterior 
Coatings” and Q/JLY J7111122A-2018 "Technical Requirements for Automotive Exterior Paint Parts". 
Compared with Q/JLY J7111122B-2019, except for editorial changes, the main technical changes are as follows: 
——修改了颜色技术要求（见4.1 一般要求4.1.1） 
Technical requirement for color is modified (see 4.1 general requirements 4.1.)  
——修改了光泽技术要求（见4.1 一般要求4.1.2） 
Technical requirement for gloss is modified (see 4.1 general requirements 4.1.2) 
——修改了膜厚技术要求（见4.1 一般要求4.1.3） 
Technical requirement for film thickness is modified (see 4.1 general requirements 4.1.3) 
——增加了附着力测试方法-安德烈切痕法和等级要求（见5.5 和4.2 表2 序号5 项） 
Increases the adhesion test methods - Andre cut marks and Level requirements (see 5.5 and table 4.2 
2 item 5) 
——增加了原标准中的规范性引用文件，增加了“ISO 6270-2 色漆和清漆 耐湿性的测定 试样置
于冷凝水环境中的试验方法” 
Increase the normative reference documents in the original standard, “Paints and varnishes - 
Determination of resistance to humidity - Procedure for exposing test specimens in condensation-water 
atmospheres” 
——修改了耐水性、耐热老化、高低温环境循环、耐湿热性和冷凝水技术要求的描述（见4.1 表1
第6-10 项） 
Changes the water resistance, heat-resistant ageing damp and hot resistance, high low temperature 
cycle, and the description of the condensed water technical requirements (see table 4.1 1 6-10) 
——删除了耐刮伤性技术要求和试验方法 
The scratch resistance technical requirements and test methods 
——修改了耐腐蚀技术要求（见4.1 表1 第15 项） 
Changes the corrosion technology requirements (see table 4.1 1 item 15) 
——删除了耐燃油试验项目中“柴油（0#）”试验试剂（见5.20） 
Deleted the fuel oil resistant test project "diesel (0 #)" test reagent (see 5.20)

1  范围  Scope 
本标准规定了汽车外饰油漆涂料类涂层的技术要求及试验方法。 
This Standard specifies the technical requirement and test method for automotive exterior paint coatings. 
本标准适用于汽车外饰油漆涂料类涂层（基地自制件除外）。 
This Standard applies to automotive exterior paint coatings (except for base self-made parts).

3.9  冷凝水试验箱  Condensate test chamber
