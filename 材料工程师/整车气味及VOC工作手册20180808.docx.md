# 一、前言

整车空气质量(VOC)控制是一项系统工程，从产品设计源头开始，到车辆批量生产，整个过程需要气味&VOC小组各成员共同协作，从设计源头开始，规避气味和VOC不合格的材料和工艺，不同开发阶段的整车和零部件的全检及改善，到量产后的高密度不间断的整车及零部件抽检，实现对整车空气质量的全生命周期管控。

本操作指导书涵盖整车开发流程（GVDP）中各阶段的整车气味&VOC管控工作，包括产品前期开发阶段（G8）、工程试制阶段（G3）、整车试生产阶段（G2-G0）和批量生产阶段（G0后）的具体工作内容，明确了气味&VOC小组各成员的职责分工，各阀点整车空气质量工作的输出物清单以及输出物签字确认审批工作。

# 二、前期开发

### 2.1 对标车型测试报告

新项目前，中心实验室根据需求对市场主要竞争车型进行整车空气质量检测，作为新项目整车空气质量控制目标的依据之一。

### 2.2 整车空气质量目标定义

项目前期G8阶段，中心实验室根据平台要求和法规要求定义新项目整车空气质量要求，包括整车VOC、整车TVOC和整车气味要求。

### 2.3 SOR材料定义

G6阶段SOR包发布前，材料工程师通过使用《材料气味及VOC规避清单》在设计之初规避气味及VOC不合格的材料及工艺，气味工程师收集内饰主要零部件的SOR包，对有异议的零件材料定义进行跟踪。

### 2.4 整车气味风险评估

工程数据发布完成后，中心实验室和整车集成科参考前车气味问题经验，评估整车气味风险；有风险的及时上升到平台，由平台进行协调。

### 2.5 零件和材料散发要求解析

TR/TA评审完成后零件的材料及工艺已基本锁定，气味工程师根据已有的经验对新项目零件和材料散发要求进行解析，设定要求。

# 三、工程试制生产和试生产

### 3.1 EP车辆气味评审记录和风险报告

通过EP车辆零件和车身结构检查，中心实验室对整车气味进行评审。

若EP车辆评审认为存在整车气味风险，需填写风险报告并告知项目管理，在项目会上进行通报；并根据决策结果推动工程进行整改。

### 3.2 ESO零部件材料散发性能整改和认可

ESO阶段零件的散发性能认可，散发性能包括气味、甲醛、雾翳、总碳、VOC和TVOC。测试零件包括乘客舱内和与乘客舱相关联的零件和材料，相关零件材料如空调箱、风道以及空腔蜡，车身胶等车身辅料。测试结果由气味工程师给出结论，详见于材料工程师的材料认可报告—ESO认可报告。

 ESO零件散发性能不合格，需要供应商分析根本原因，采取有效措施来加以改进；供应商需要提交8D整改报告，气味工程师对8D整改报告的有效性进行评估，评估合格的接受整改样件，重新开始认可。

### 3.3 PPV造车过程记录

中心实验室对PPV造车过程确认，排除整车的气味和VOC风险。

### 3.4 PPAP零部件材料散发性能整改和认可

PPAP阶段零件的散发性能认可，认可零件是ESO阶段检测散发性能条件认可的零件，其中气味性能需全部重新检测。

PPAP零件散发性能不合格，需要供应商分析根本原因，采取有效措施来加以改进；供应商需要提交8D整改报告，气味工程师对8D整改报告的有效性进行评估，评估合格的接受整改样件，重新开始认可。

### 3.5 PPV-P整车空气质量及气味检测、整改。

中心实验室在PPV、PP和P阶段对整车进行气味和VOC检测，根据检测结果来评估整车气味和VOC散发的风险。整车检测结果不合格的，气味工程师对整车进行溯源，对主要零部件进行筛选，找到不合格的根本原因，并跟踪落实整改措施，直至满足要求。

# 四、批量生产

### 4.1季度零件VOC抽检报告

实验室每年初制定当年的VOC抽检季度抽检计划，每季度对重点零件抽检VOC性能。

### 4.2季度零件气味抽检记录表

实验室每年初制定当年的气味抽检季度抽检计划，每季度对重点零件抽检气味性能。

### 4.3整车空气质量抽检报告

按计划每年对所有车型进行整车空气质量抽检，抽检频率与整车销售成正比。

### 4.4现场实验室日常巡检表

现场实验室每日对整车和零件进行气味抽检。

### 4.5量产不合格零件处理和整改

实验室通过《实物质量缺陷确认单》对散发性能不合格的零件供应商进行处理。

零件散发性能不合格，需要供应商分析根本原因，采取有效措施来加以改进；供应商需要提交8D整改报告，气味工程师对8D整改报告的有效性进行评估，评估合格的接受整改样件，重新开始认可。

# 五、特殊车辆

### 5.1 零件及整车特殊工艺记录表

对展车、政府用车等特殊车辆，中心实验室接受公司内其他部门的委托进行整车检测和零部件的处理。

### 5.2 整车空气质量和气味测试报告

对展车、政府用车等特殊车辆进行整车空气质量和气味检测。

### 5.3 送检整车状态确认清单

认证用车辆送检前，必须由中心实验室来确认车辆状态。

### 5.4 整车空气质量和气味测试报告

认证用车辆送检前，由中心实验室进行整车空气质量和气味检测，确认合格后才能送检。
