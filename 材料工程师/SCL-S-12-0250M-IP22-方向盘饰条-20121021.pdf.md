Ⅱ. 结论与建议 Conclusions and suggestions 
结论：1）失效件上油漆的膜厚低于TDS 要求（但这不是导致磨损的主因）。 
            2）提高膜厚（即使远远高于TDS 要求），仍然不能避免磨损，唯一和低膜厚区别，是今后磨损直至基材的时间长短不同。 
3）失效件的产生主要原因是：使用了不恰当的油漆体系（单层金属漆的耐磨损性能并不好，不适用于高磨损强度的地方），零件的耐
磨损性能不能满足其实际使用需要。（正确的做法是加一层清漆）。 
     建议： 1）充分考虑零件的使用环境，选择合适的材料。 
2）充分考虑零件的使用环境，选择合适的试验项目去考核。 
    Conclusion： 1）The film thickness of the failure part was lower than the min. demand of the TDS(but this is not the root cause). 
                 2） Even though the film thickness is increased far more than that of TDS, abrasion trace is still existed. The high thickness 
only can make the substrate to be seen later.  
3） The main cause is:  the unsuitable coating system was used （actually one top clear coat should be added） 
Suggestion:  
1）The part’s  using environment should be fully considered, then the suitable material should be chosen （the unsuitable coating system 
was chosen in this case） 
2）The part’s  using environment should be fully considered, then the suitable test item should be chosen.

小结：1）失效件和涂装饰板均为单层漆，牌号VPCH02726，为聚氨酯双组份金属效果漆。一般来说金属效果漆的耐磨损性能一般，
不适用于有较高耐磨损要求的区域，如果需要获得较高的耐磨损性能，需要在其表面再喷涂一层清漆。 
2）不管是失效件还涂装饰板，都不能通过2000 次的耐磨损试验。即不管该牌号油漆的膜厚高低，都不适用于高耐磨损区域。因此，
失效件的膜厚达不到TDS 要求，不是导致磨损的直接原因，即使漆膜再厚，表面磨损仍不能避免，这是含有金属粉末的油漆本身性
能决定的。但是漆膜厚度不同，会导致磨损至基材的时间长短不同。 
3）在设计中，应绝对避免把单层金属漆直接用于高耐磨负载的零件上。
