## 一、光致变色涂料（Photochromic Coating）

### 原理

某些化合物在特定波长光照下，**分子结构发生可逆变化**（如顺反异构、开环/闭环反应），导致对光的吸收峰值改变，从而呈现不同颜色；撤除光照后，分子恢复原状，颜色复原。

**典型变色材料体系：**

| 类型 | 代表物质 | 变色特点 |
|------|---------|---------|
| 螺吡喃类 | Spiropyran | 无色→有色，紫外触发 |
| 螺恶嗪类 | Spiropyran analog | 稳定性更好 |
| 二芳基乙烯类 | Diarylethene | 抗疲劳性强，循环次数高 |
| 俘精酸酐类 | Fulgide | 耐候性优良 |

### 工艺要点

- 颜料需充分分散于透明树脂体系（溶剂型/水性均可）
- 搭配 **UV 吸收剂 + 光稳定剂** 防止光疲劳，延长循环寿命
- 涂层结构：底漆 → 光变层 → 透明面漆（保护层）
- 可与 **光干涉颜料**（MIO/云母系）配合使用，增强珠光/金属质感

### 汽车装饰件应用现状

| 应用部位 | 形式 | 成熟度 |
|---------|------|--------|
| 车身外观改色漆 | 整车涂装 | 概念车/高端车型 |
| 车标 / Logo | 局部涂层 | 小批量 |
| 后视镜壳 | 变色效果 | 小批量 |
| 车窗玻璃 | 镀膜或贴膜 | 已实用 |

> ⚠️ 光致变色在汽车外饰的应用**受限较多**：日光下颜色变化受纬度/季节/天气影响大，且颜料耐 UV 老化性能仍是瓶颈，目前多见于概念车或售后改色膜。

## 二、电致变色涂料（Electrochromic Coating）

### 原理

在 **1-5V 直流电场** 作用下，电致变色材料发生 **可逆氧化还原反应**，伴随 H⁺、Li⁺、OH⁻ 等离子嵌入/脱嵌，材料的吸收/反射光谱特性改变，从而实现颜色变化。

**核心反应（无机系代表 — 氧化钨）：**

```
WO₃（透明/浅色） + xH⁺ + xe⁻ ⇌ HₓWO₃（深蓝/深色）
```

**有机系代表（聚苯胺 PANI）：**

```
PANI（淡黄色） ⇌ PANI⁺（绿色） ⇌ PANI²⁺（蓝紫色）
```

**其他重要材料体系：**

- 氧化镍（NiO）：对电极/离子存储层
- 紫罗精（Viologen）：有机电致变色材料，颜色切换快
- 酞菁：金属有机络合物，稳定性好

### 五层结构工艺

```
透明导电层（ITO / ATO 玻璃）
        ↓
电致变色层（WO₃ / PANI / 紫罗精）
        ↓
离子传导层（电解质层）
        ↓
离子存储层（对电极，如 NiO）
        ↓
保护层（SiO₂ / 树脂封装）
```

**工艺路线对比：**

| 工艺 | 原理 | 优势 | 劣势 |
|------|------|------|------|
| 磁控溅射 | 物理气相沉积 | 膜层致密，质量高 | 设备贵，成本高 |
| 溶胶-凝胶法 | 化学液相沉积 | 适合大面积，成本低 | 均匀性稍差 |
| 湿法涂布（卷对卷） | 柔性膜涂布 | 可做曲面，产能高 | 需解决柔性问题 |
| CVD | 化学气相沉积 | 附着力强 | 工艺复杂 |

> **光羿科技的突破**：采用 **湿法涂布** 路线，第一次让电致变色技术具备低成本、高规格、规模化量产的可能性，解决了传统工艺无法适配汽车大尺寸曲面天幕的难题。

### 核心性能参数

| 参数 | 典型值 | 说明 |
|------|--------|------|
| 响应时间 | 10-60 秒 | EC 技术相对较慢 |
| 变色循环 | >10⁵ 次 | 寿命长 |
| 透光率范围 | 10%-80% | 可调范围大 |
| 工作电压 | 1-5V DC | 低压安全 |
| 能耗 | <1W/m² | 极低能耗 |

## 三、汽车装饰件应用 — 详细梳理

### 🚗 外饰

#### ① EC 电致变色天幕（最大应用场景）

- EC 是目前新能源汽车使用最多的调光技术路线
- 第三代柔性 EC 技术可适配 **大尺寸曲面造型**（曲率半径可达 5mm）
- **技术路线对比：**
  - EC 调光慢但均匀且可曲面
  - PDLC 快但不适配大弧度玻璃
  - LC/DLC 染料液晶速度快，但不适配大面积弧线玻璃

**代表车型：**

| 车型 | 供应商 | 技术 |
|------|--------|------|
| 蔚来 EC7 | 自研 | 第三代柔性 EC，总能量通过率最低 12% |
| 极氪 001 | 光羿科技 | EC 光感天幕 |
| 埃安 S Plus | 光羿科技 | EC 光感天幕 |
| 比亚迪海豹 | 供应商 | EC 光感天幕 |
| 小米 YU7 | 光羿科技 | EC 天幕，0.1 秒级响应 |
| 宝马 iX | — | PDLC 调光天幕（标配） |
| 智己 L6 | — | PDLC 调光天幕（标配） |
| 极氪 MIX | — | PDLC 调光天幕 |
| 路特斯 EMEYA | — | PDLC 调光天幕（S+ 标配） |

#### ② 自动防眩目后视镜

- 成熟应用，EC 材料（如氧化钨）在后视镜领域已商业化多年
- 凯迪拉克、奔驰等品牌早已采用
- Gentex 是全球最大后视镜供应商，EC 后视镜核心厂商

#### ③ 智能车标 / 发光格栅

- 结合 EC 变色层，可实现动态发光效果
- 2025 年小米 YU7 等新车型的亮点配置

### 🏠 内饰

- **内饰氛围灯/面板**：EC 变色膜用于仪表板、门板装饰
- **隐私玻璃**：一键切换透明/磨砂状态
- 整体仍处于 **概念阶段**，量产较少

### 驱动因素

**利好：**

- 🏭 **新能源汽车大尺寸天幕** 成为标配，带动 EC 调光需求爆发
- 🎨 **个性化/智能化** 消费趋势，车企差异化竞争需求
- 🌡️ **节能需求**：EC 天幕可降低空调能耗 30% 以上
- 📈 **技术突破**：柔性 EC 涂布工艺解决曲面适配问题，成本持续下降
- 🤖 **智能驾驶**：自动驾驶时代，车窗/天幕成为"第三块屏幕"，变色膜是交互媒介

**制约：**

- EC 变色响应速度（10-60s）仍是痛点，PDLC 路线分流部分市场
- 光致变色涂料在汽车外饰的 **耐候性/耐久性** 仍未完全解决
- 量产成本高，OEM 接受度需时间

### 趋势判断

```
短期（2025-2027）：EC 天幕成为 20 万+新能源车型标配
中期（2027-2030）：EC 技术向侧窗/后窗延伸，内饰氛围件开始量产
长期（2030+）：光致变色外饰漆 + EC 智能玻璃融合，"变色汽车"走向大众
```

| 对比维度 | 光致变色 | 电致变色（EC） |
|---------|---------|--------------|
