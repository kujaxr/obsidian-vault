# SCL-L-23-0273M-遥控钥匙-徐睿.docx

   测试报告  Test report           

Ⅰ.任务描述 Task description

实验室受委托对显亮（昆山）汽车配件有限公司提供的贴膜（遥控钥匙）进行测试。该钥匙LOGO装饰片（PMMA）在总装现场被抱怨频繁划伤，故显亮申请表面贴保护膜，SQE委托实验室对贴膜后对表面影响进行摸底测试。贴膜种类与牌号未提供。

Ⅱ. 结论与建议 Conclusions and suggestions

1，对贴膜后的钥匙按照SMTC 5 400 005B进行8cyc/96h环境交变试验，期间48h表面无变化，96h试验完成后揭开贴膜表面出现气泡状残胶，用湿布无法完全擦除；

2，对贴膜后的钥匙进行80℃@240h高温存放，试验后表面无明显变化。

综合以上结果，钥匙上贴膜保留时间不建议超过6个月。
