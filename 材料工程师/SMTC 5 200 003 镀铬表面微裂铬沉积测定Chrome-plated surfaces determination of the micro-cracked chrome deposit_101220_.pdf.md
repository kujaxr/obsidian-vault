1 范围 
本标准规定了测定镀铬表面微裂铬沉积的
方法。  
本标准适用于测定镀铬表面微裂铬沉积。 
1  Scope  
This standard specifies a method for determining 
the micro-cracked chrome deposit on chrome plated 
surfaces. 
This standard adopts to determine the mi-
cro-cracked chrome deposit on chrome plated sur-
faces.  
2 测试  
2 
Test  
2.1 测试设备及仪器 
测定微裂铬沉积需用 100-200 倍反射光
显微镜，在具有明场基础上，该显微镜还必须
具有暗场功能。为了量化铬微裂密度，使用的
显微镜的目镜必须具有蚀刻测量线，或者必须
具备可校准的图像记录设备。 
3.1  Testing equipment  
A 100 x to 200 x reflected light microscope that 
has a dark field capacity in addition to the normal 
bright field capacity is required in order to determine 
the micro-cracked chrome deposit. 
    In order to count the micro-cracked chrome den-
sity, the microscope must either have etched meas-
urement lines in the eyepiece or an image recording 
device that can be calibrated. 
3.2  浸蚀溶液 
有些情况需要用到浸蚀技术来对微裂铬沉
积进行评估。水状浸蚀溶液中含有浓度为 85.0 
g/L 的氢氧化钠(NaOH) 和浓度为 53.0 g/L 的 
K3 (Fe(CN) 6)。 
3.2  Etching solution 
In order to be able to evaluate the micro-cracked 
chrome deposit, etching is required in some cases. 
The aqueous etching solution contains 85.0 g/L 
NaOH and 53.0 g/L K3 (Fe (CN) 6). 
3.3  测试步骤 
3.3  Test parameters 
3.3.1  先用酒精或丙酮清洗试样，然后将样品
置于显微镜下，通过放大系数为 100 倍的暗场
来进行观察。如果微裂铬沉积清晰可见，就可
以使用目镜的刻度线对每厘米的裂纹进行计
数，或者通过可校准的图像记录设备进行计数。
3.3.1  The specimen part is cleaned with alcohol or 
acetone and viewed at 100 x enlargement underthe 
microscope using the dark field. If the micro-cracked 
chrome deposit is clearly dentifiable. The number of 
cracks per cm is counted using the measurement 
lines in the eyepiece or by means of a calibrated re-
cording. 
3.3.2  如果裂纹网格不能被清晰的显示出来，
那必须对样本进行浸蚀处理，再拿到显微镜下
重新观测。根据浸蚀溶液的状态，浸蚀处理过
程需要 10 min 到 20 min。如果在 20 min 的
浸蚀处理之后，仍然无法清晰辨别微裂铬沉积，
则将使用过的浸蚀溶液丢弃，重新配制新的浸
蚀溶液并重复浸蚀处理。 
3.3.2  If the crack network cannot be perfectly iden-
tified, the part must be etched and then viewed under 
the microscope. The etching duration is, depending 
on the condition of the etching solution, approx. 10 to 
20 min. If a micro-cracked chrome deposit is still not 
clearly identifiable after 20 min., the etching solution 
is discarded and made up again and the etching 
procedure is repeated. 
3.3.3  或者，也可以用库仑测厚仪，通过阳极
浸蚀法来进行这一操作，浸在浓度为 10%、通
入 0.5 A 电流的氢氧化钠 (NaOH)  溶液 5 s。
3.3.3  Alternatively, anodic etching with a coulosc
ope for 5s at 0.5 A in 10% NaOH is permitted.  
3.3.4  最好将显微镜放大系数设置为 100 
倍，通过明视野来对经过浸蚀处理的样本进行
观测。如果无法清晰观测微裂网格，则必须将
显微放大倍数调至 200 倍进行评估。 
3.3.4  Etched parts should preferably be evaluated 
at 100 x enlargement in the bright field. If the mi-
cro-crack network cannot be identified perfectly with 
this method, 200 x enlargement must be used for the 
evaluation.

structure (e.g. incomplete, regular) are to be indi-
cated in the test report. Example: 600 to 800
