Ⅱ. 结论与建议 Conclusions and suggestions 
结论： 1．人工修边过多。 2．胶水喷涂不均匀。3．码钉数量少 
Conclusions: 1. The deburring of PVC is too much. 
2. Glue spray is not uniform.  
3. The number of code nails are small. 
建议： 1. 增加修剪定位工装。2.严格控制胶水工艺。 3.增加码钉 
Suggestions: 1. Increase the trim location tooling. 2. Control the glue process strictly. 3. Add code nails.

III. 宏观分析 Macroscopic observation 
据售后人员反馈，从2016 年7 月到目前为止，中控台扶手失效共计24 例，具体失效信息见表
1 所示。所有失效件失效模式一致，均表现为扶手表面PVC 皮脱出。 
将正常件及失效件拆解对比，发现失效件PVC 皮包覆面积小，且扶手本体表面胶水不均匀，
码钉装订位置有偏差，如图2 所示。正常件在失效位置处PVC 包覆面积大，码钉固定在PVC 包边
的中心处。

对供应商的工艺过程进行检查，发现以下问题：扶手骨架未经清洁直接喷胶，影响表皮粘接力；
喷胶方式为人工喷涂，其喷涂不均匀且无法控制喷胶量；胶水的使用要求是开盖后48h 内使用，但
供应商对胶水使用无监控无跟踪；扶手PVC 表皮包好后需要人工修边，此过程不可控，修边过多会
直接导致后期的PVC 表皮脱出。
