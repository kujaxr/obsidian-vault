# 一、前言

整车腐蚀是一项系统工程，防腐工作的开展，从产品设计源头开始，到车辆批量生产，整个过程需要防腐小组各成员共同协作，从设计源头开始，建立涵盖结构、材料及工艺等方面的控制体系，通过产品结构设计、选材控制及生产工艺控制，实现对整防腐质量的有效管控。

本操作指导书涵盖整车开发流程（GVDP）中各阶段的整车腐蚀工作，包括产品前期开发（G5前）、工程试制造车（G5）、生产线造车工艺验证（G4-G3）、批量试生产（G2）和批量抽检（G0后）的具体防腐工作内容，明确了防腐小组各成员的职责分工，各阀点整车腐蚀工作的输出物清单以及输出物签字确认审批工作。

# 二、前期开发(TG2)

前期开发防腐工作，根据车型定位对整车防腐目标进行定义，在TG2数模发布前的数模进行电泳工艺分析、数模防腐检查和涂胶定义，从设计前期对整车防腐设计把关，规避相关腐蚀问题。

批准：                                            

### 2.1整车防腐目标定义

1、工作内容：

根据项目车型分类、市场定位、销售对象、销售区域、产品质量目标等信息确定整车开发防腐目标定义，作为后续整车防腐工作开展依据。

2、输出物模板见附件1

### 2.2防腐同步工程

在TG2工程数据发布前，依据车身设计经验、整车腐蚀案例等相关经验，对车身数模进行检查，包括三方面内容：电泳工艺可行性、数模防腐检查以及整车涂胶PAM，规避车身设计中的腐蚀问题。

1、电泳工艺可行性主要检查车身数模中的工艺孔、漏液孔、漏液筋、排气孔、钣金间隙是否满足防腐设计要求，分析车身结构中潜在电泳不良区域并提出优化方案；

2、数模防腐检查是针对整车结构设计、镀锌板选用、涂胶定义、零件布置等方面进行检查，规避潜在防腐设计问题；

3、整车涂胶PAM，对车身各区域涂胶种类、涂胶要求进行定义，包括细密封、粗密封、底部抗石击胶、结构胶、折边胶、密封胶、减震胶、膨胀块、阻尼垫&增强垫。

2.2.1电泳工艺可行性

1、工作内容：

在TG2工程数据发布前，检查车身钣金电泳工艺孔、漏液孔、漏液筋、排气孔以及多层板之间间隙，对车身钣金电泳可油漆性进行评估，避免兜液、积气、电泳不良区域，确保整车内腔电泳漆膜厚≥10μm。

2、检查清单：

3、输出物模板见附件2

2.2.2数模防腐检查

1、工作内容：

数模防腐检查针对以下几个方面：

1）整车结构设计方面：重点区域钣金搭接和错边、两厢车车顶尾部线束孔凸台，空腔排气孔是否设计在最高点，排液孔是否设计在最低点等；

2）镀锌板选用方面：腐蚀环境恶劣区域钣金是否采用镀锌板，包括四门两盖内外板、翼子板及支架、侧围、前导水槽及支架、后导水槽、后保支架、金属加油口盖等；

3）涂胶定义方面：外露车身钣金切边是否PVC防护、车顶和门加强板减震胶是否断胶避免电泳不良、底部钣金迎风面和石击区域是否有底喷覆盖等；

4）零件布置方面：脱钩布置是否远离热源、发盖支撑杆支架是否布置在水道上、尾门气弹簧支架布置在水道上、落水槽饰板是否遮蔽前围板切边等，规避潜在防腐设计问题

# 三、工程试制造车（EP）

工程试制造车阶段，整车防腐工作主要分三块方面的内容，造车跟踪，车身评价以及试验。本阶段资源需求为一台电泳车身和一台整车，电泳车身用于盐雾试验，整车用于整车腐蚀试验。EP工程车的车身焊接是在PPO进行，用于整车腐蚀试验的车身涂装在生产工厂进行，总装会返回PPO车间。造车过程需进行全程跟踪，从小分拼钣金表面检查，车身胶涂胶，车身焊接，涂装，总装跟踪记录，为腐蚀问题追溯提供参考。白车身、电泳车身进行防腐检查，并进行防腐试验，确保车身结构设计满足防腐要求。

### 3.4整车试验

腐蚀车造车完成后对试验车进行发车前检查，然后签发送试验场进行整车强化腐蚀试验，对试验过程进行记录。试验过程中进行2腐蚀年、5腐蚀年和10腐蚀年拆解评估检查，并出具相关报告，跟进问题整改至问题关闭。

3.4.1整车强化腐蚀试验车签发

1、工作内容：

腐蚀试验车在运往试验场前，需汇总造车过程的材料，防腐小组对整车状态进行检查评估，车身钣金质量及涂胶工艺、油漆膜厚、零件状态等，确认车辆可用于整车腐蚀试验，防腐小组各成员会签后发车去试验场试验。

3.4.2  0-check检查

1、工作内容：

腐蚀试验车在进行整车腐蚀试验前需对整车状态进行记录拍照，以便后续腐蚀问题追溯。检查内容包括整车油漆状态，重点关注钣金切边涂胶情况，车身胶可见区域有无开裂情况，零件装配损伤情况、有无磕碰，干涉、硬接触以及表面涂层损伤锈蚀状态记录，还有部分改装件、缺失件检查记录。

3.4.3  2腐蚀年检查

1、工作内容：

腐蚀试验车完成2腐蚀年后，防腐小组会对整车及零部件的腐蚀状态进行检查，依据SMTC 3 0H4 001-2013《整车腐蚀性能规范》对零件腐蚀状态进行评价。重点检查底盘、发动机舱区域零部件，同时关注车身钣金及附件、乘客舱零件、管路接头、车身胶状态等，评估腐蚀是否超出标准要求。底盘高可见度零部件腐蚀等级不得低于7级，中低可见度零部件腐蚀等级不得低于6级；发动机舱高可见度零部件腐蚀等级不得低于8级，中低可见度零部件腐蚀等级不得低于6级。对评估不满足腐蚀要求的零件进行整改。

3.4.4  5腐蚀年检查

1、工作内容：

腐蚀试验车完成5腐蚀年后，防腐小组会对整车及零部件的腐蚀状态进行检查，依据SMTC 3 0H4 001-2013《整车腐蚀性能规范》对零件腐蚀状态进行评价。重点检查车身钣金及附件、内外饰零部件、安全件、车身胶状态、车身划线部位腐蚀扩展宽度等，评估腐蚀是否超出标准要求。车身外表面高中可见度钣金不得低于10级，零部件级紧固件不得低于9级，车身外表面低可见度钣金不得低于8级，零部件及紧固件不得低于7级；车身次表面高中可见度钣金不得低于10级，零部件及紧固件不得低于8级，车身次表面低可见度钣金不得低于8级，零部件及紧固件不得低于7级。乘客舱高中可见零部件腐蚀等级不得低于8级，低可见度零部件腐蚀等级不得低于3级。底部安全件，腐蚀等级不得低于7级。车身划线钣金腐蚀单边扩展宽度：镀锌板≤3mm，冷轧板≤5mm。对评估不满足腐蚀要求的零件进行整改。

2、检查清单：

3、输出物模板见附件14

3.4.5  10腐蚀年检查

1、工作内容：

腐蚀试验车完成10腐蚀年后，防腐小组会对整车及零部件的腐蚀状态进行检查，评估整车零部件功能是否因腐蚀导致失效，然后对整车进行拆解。检查车身胶状态是否开裂、剥离，性能是否满足要求；检查车身空腔锈蚀状态及评估钣金是否存在锈穿风险，钣金表面有金属损失，锈蚀物可剥离即认定有锈穿风险。对评估不满足车身胶性能要求、不满足腐蚀要求钣金及功能失效零部件进行整改。

# 四、生产线造车工艺验证（PPV-PP）

生产线造车工艺验证阶段，车身及总装在工厂进行，生产工艺调试验证，这个阶段整车腐蚀工作主要是造车工艺验证、防腐检查及整车试验。本阶段资源需要两台电泳车身（PPV和PP各一台）和一台整车。电泳车身拆解，验证车身电泳工艺、涂胶质量，搭载红药水试验，验证折边胶工艺。整车用于整车腐蚀试验，检验生产线生产工艺，验证EP腐蚀车试验中发现的腐蚀问题。本阶段整车腐蚀试验根据EP腐蚀车试验进展结果以及项目开发进度确定是否进行。本阶段整车腐蚀工作用于车身ESO认可。

批准：                                            

### 4.1生产线造车过程记录

1、工作内容：

造车过程涵盖钣金涂车身胶、白车身钣金质量检查、涂装前处理、电泳、密封、中涂、面漆、喷蜡、总装等。整个制造过程需要跟踪，记录制造过程及各工序的工艺参数，为后续工艺调整以及腐蚀问题追根溯源提供依据。

2、检查清单：

3、输出物模板见附件6

### 4.2  生产线造车防腐检查

工艺验证和预试生产阶段的产品已安排在工厂进行，调试生产工艺和审核制造成熟度。在PPV和PP阶段拆解黑车身，检查车身钣金电泳质量验证电泳工艺，检查车身胶质量验证涂胶工艺，红药水试验验证折边工艺；通过整车评审，验证实车装配工艺。

4.2.1  车身胶拆解

1、工作内容：

2、检查清单

3、输出物模板见附件10

4.2.2  电泳车身拆解

1、工作内容：

在PPV阶段拆解一台黑车身，检查车身电泳漆膜厚、电泳漆质量、空腔膜厚、车身蜡质量、底喷胶质量，重点关注的区域PVC涂胶质量等。PPV阶段检查的电泳&涂PVC问题，在PP阶段对进行整改并拆解验证。

2、检查清单

3、输出物模板见附件16

2、检查清单

3、输出物模板见附件17

4.2.4  车身防腐AUDIT检查

3、输出物模板见附件8

### 4.3整车试验

PPV&PP试验车造车完成后对腐蚀试验车进行发车前检查，然后签发送试验场进行整车强化腐蚀试验，对试验过程进行记录。试验过程中进行2腐蚀年、5腐蚀年和10腐蚀年拆解评估检查，并出具相关报告，跟进问题整改至问题关闭。

2、输出物模板见附件11

腐蚀试验车完成5腐蚀年后，防腐小组会对整车及零部件的腐蚀状态进行检查，依据SMTC 3 0H4 001-2013《整车腐蚀性能规范》对零件腐蚀状态进行评价。重点检查车身钣金及附件、内外饰零部件、安全件、车身胶状态、车身划线部位腐蚀扩展宽度等，评估腐蚀是否超出标准要求。车身外表面高中可见度钣金不得低于10级，零部件级紧固件不得低于9级，车身外表面低可见度钣金不得低于8级，零部件及紧固件不得低于7级；车身次表面高中可见度钣金不得低于10级，零部件及紧固件不得低于8级，车身次表面低可见度钣金不得低于8级，零部件及紧固件不得低于7级。乘客舱高中可见零部件腐蚀等级不得低于8级，低可见度零部件腐蚀等级不得低于3级。底部安全件，腐蚀等级不得低于7级。车身划线钣金腐蚀单边扩展宽度：镀锌板≤3mm，冷轧板≤5mm。对评估不满足腐蚀要求的零件进行整改。

| 整车涂胶定义 |

| 1、细密封 |

| 一般用于零件装配面，顾客在使用过程中可见位置，在起到密封、防腐功能的同时具备一定的美化功能PVC细密封。PVC细密封按工艺方式一般分为柱胶与刮胶。 |

| 1）柱胶 |

| 要求：涂胶宽度5-9mm，涂胶厚度2-3mm，涂胶光滑连续，无野胶、缩孔、开裂等外观缺陷，单条胶小于5mm的气泡不超过2个，且间距大于10cm，气泡不可开裂。 |

| 区域：四门及两盖钣金包边 |

| 2）刮胶 |

| 要求：在涂抹或者弄平操作时必须保证不要露出要保护的板材边缘，并且必须要有足够厚的聚氯乙烯涂覆层完全盖住板材。 |

| 区域：一般用于有装配要求的部位，如尾灯安装区域，前风挡安装区域；或是对外观有要求的部位，如后导水槽钣金边缘，顶盖、侧围、后导水槽交接区域等。 |

| 2、粗密封 |

| 要求：涂胶宽度15-30mm，同一胶条打胶宽度需一致，涂胶厚度2-3mm，涂胶边角平整，无溢胶、堆胶、残胶等外观缺陷，毛刷枪打胶后，毛笔刷胶宽度小于30mm，不允许在车身钣金上清理毛刷上的多余胶，气泡不可开裂。 |

| 区域：一般用于被内外饰覆盖的部位，无外观要求，如前围、地板、纵梁延伸钣等钣金缝隙处。 |

| 3、底部抗石击胶 |

| 要求：涂胶厚度为500-1000um，表面涂胶均匀，无堆胶。避让螺纹孔与安装孔。 |

| 区域：前底部、后地板及轮罩容易受到石击区域。 |

| 4、结构胶 |

| 要求：一般涂胶直径在2-3mm，固化后的胶接面（痕）的宽度不小于10mm；如果涂胶在搭接边处，则要求胶条中心离搭接边边缘距离不小于7mm；如果胶涂在较薄的外覆盖件处，则要求胶的位置在外覆盖件的翻边或加强筋等具有较强结构强度处，距离加强结构边缘5mm以内，同时胶条中心离搭接边缘距离不小于7mm。 |

| 区域：由钣金工程师根据结构强度要求定义涂胶区域 |

| 5、折边胶 |

| 要求：直径通常为2-3mm，接触面A的涂敷宽度不小于10mm，接触面B的涂敷率不小于20% |

| 区域：用于车门、发动机罩、后备箱翻边处。 |

| 6、密封胶 |

| 要求：一般涂胶直径在2-3mm，固化后的胶接面（痕）的宽度不小于10mm；如果涂胶在搭接边处，则要求胶条中心离搭接边边缘距离不小于7mm。 |

| 区域：用于直接或间接与乘客舱连接的钣金缝隙，起到密封、保护焊点等作用。 |

| 7、车身减震胶 |

| 要求：胶的边缘距离涂胶所在位置边缘不小于5mm；如果涂胶为长条状，通常涂胶直径D应比涂胶所在位置的两零件间隙a大3-4mm。 |

| 区域：外覆盖件与加强件之间的连接，起到减少外钣震动的作用，如四门两盖加强板与外板之间，车顶加强板与顶盖之间，加油口盖与侧围之间。 |

| 8、膨胀块 |

| 要求：膨胀块结构:根据位置和空腔尺寸决定2D、3D方案选择,钣金间隙超过28mm，建议设计3D。3D件EVA与钣金间隙在常规位置建议为4-6mm，在尖角处建议为1.5mm。烧焊离膨胀块至少30mm。离各种安装孔距离≥30mm，保证发泡后不堵孔。 |

| 区域：A柱、B柱、C柱、门槛和轮罩空腔。 |

| 9、阻尼垫&增强垫 |

| 要求：阻尼垫贴附钣金阻尼因子在0-32℃要求≥0.1，其中20℃要求≥0.11。厚度一般选择2-3mm。增强垫增强倍率≥2，钣金受力150N时，变形量≦9.5mm，钣金无折印，厚度一般为0.9-2mm。 |

| 区域：前地板、后地板和车顶。 |

| 整车涂胶定义 |

| 1、细密封 |

| 一般用于零件装配面，顾客在使用过程中可见位置，在起到密封、防腐功能的同时具备一定的美化功能PVC细密封。PVC细密封按工艺方式一般分为柱胶与刮胶。 |

| 1）柱胶 |

| 要求：涂胶宽度5-9mm，涂胶厚度2-3mm，涂胶光滑连续，无野胶、缩孔、开裂等外观缺陷，单条胶小于5mm的气泡不超过2个，且间距大于10cm，气泡不可开裂。 |

| 区域：四门及两盖钣金包边 |

| 2）刮胶 |

| 要求：在涂抹或者弄平操作时必须保证不要露出要保护的板材边缘，并且必须要有足够厚的聚氯乙烯涂覆层完全盖住板材。 |

| 区域：一般用于有装配要求的部位，如尾灯安装区域，前风挡安装区域；或是对外观有要求的部位，如后导水槽钣金边缘，顶盖、侧围、后导水槽交接区域等。 |

| 2、粗密封 |

| 要求：涂胶宽度15-30mm，同一胶条打胶宽度需一致，涂胶厚度2-3mm，涂胶边角平整，无溢胶、堆胶、残胶等外观缺陷，毛刷枪打胶后，毛笔刷胶宽度小于30mm，不允许在车身钣金上清理毛刷上的多余胶，气泡不可开裂。 |

| 区域：一般用于被内外饰覆盖的部位，无外观要求，如前围、地板、纵梁延伸钣等钣金缝隙处。 |

| 3、底部抗石击胶 |

| 要求：涂胶厚度为500-1000um，表面涂胶均匀，无堆胶。避让螺纹孔与安装孔。 |

| 区域：前底部、后地板及轮罩容易受到石击区域。 |

| 4、结构胶 |

| 要求：一般涂胶直径在2-3mm，固化后的胶接面（痕）的宽度不小于10mm；如果涂胶在搭接边处，则要求胶条中心离搭接边边缘距离不小于7mm；如果胶涂在较薄的外覆盖件处，则要求胶的位置在外覆盖件的翻边或加强筋等具有较强结构强度处，距离加强结构边缘5mm以内，同时胶条中心离搭接边缘距离不小于7mm。 |

| 区域：由钣金工程师根据结构强度要求定义涂胶区域 |

| 5、折边胶 |

| 要求：直径通常为2-3mm，接触面A的涂敷宽度不小于10mm，接触面B的涂敷率不小于20% |

| 区域：用于车门、发动机罩、后备箱翻边处。 |

| 6、密封胶 |

| 要求：一般涂胶直径在2-3mm，固化后的胶接面（痕）的宽度不小于10mm；如果涂胶在搭接边处，则要求胶条中心离搭接边边缘距离不小于7mm。 |

| 区域：用于直接或间接与乘客舱连接的钣金缝隙，起到密封、保护焊点等作用。 |

| 7、车身减震胶 |

| 要求：胶的边缘距离涂胶所在位置边缘不小于5mm；如果涂胶为长条状，通常涂胶直径D应比涂胶所在位置的两零件间隙a大3-4mm。 |

| 区域：外覆盖件与加强件之间的连接，起到减少外钣震动的作用，如四门两盖加强板与外板之间，车顶加强板与顶盖之间，加油口盖与侧围之间。 |

| 8、膨胀块 |

| 要求：膨胀块结构:根据位置和空腔尺寸决定2D、3D方案选择,钣金间隙超过28mm，建议设计3D。3D件EVA与钣金间隙在常规位置建议为4-6mm，在尖角处建议为1.5mm。烧焊离膨胀块至少30mm。离各种安装孔距离≥30mm，保证发泡后不堵孔。 |

| 区域：A柱、B柱、C柱、门槛和轮罩空腔。 |

| 9、阻尼垫&增强垫 |

| 要求：阻尼垫贴附钣金阻尼因子在0-32℃要求≥0.1，其中20℃要求≥0.11。厚度一般选择2-3mm。增强垫增强倍率≥2，钣金受力150N时，变形量≦9.5mm，钣金无折印，厚度一般为0.9-2mm。 |

| 区域：前地板、后地板和车顶。 |
