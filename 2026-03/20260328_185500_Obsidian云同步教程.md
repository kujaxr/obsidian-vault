## 一、准备工作：5 分钟创建 GitHub 云端仓库

1. 打开 [GitHub 官网](https://github.com)，注册/登录账号
2. 点击右上角 **New repository**（新建仓库）
   - Repository name：随便起，比如 `my-obsidian-vault`
   - Visibility 务必选 **Private（私有）** —— 这是重中之重！
   - 点击 Create repository
3. 下载 **GitHub Desktop** 官方客户端，安装后登录

---

## 二、电脑端基础配置：克隆仓库 + Obsidian 打开

1. **克隆仓库**
   - GitHub Desktop → File → Clone repository
   - 选择本地保存路径（如 `D:\Obsidian-Vault`）→ Clone

2. **Obsidian 打开**
   - Obsidian → Open folder as vault → 选中刚才克隆的文件夹

3. **配置 .gitignore（防止冲突）**
   
   在仓库根目录新建 `.gitignore` 文件，内容：
   ```
   .obsidian/workspace.json
   .obsidian/workspace-mobile.json
   .obsidian/appearance.json
   ```

4. **首次提交**
   - GitHub Desktop → Commit → Push

---

## 三、电脑端自动同步：Obsidian Git 插件

1. **安装插件**
   - Obsidian 设置 → 第三方插件 → 关闭安全模式 → 搜索 `git` → 安装

2. **配置自动同步**
   - Auto commit and sync after stopping file edits：**1 分钟**
   - Pull on startup：**开启**
   - 设置完后，写笔记时完全不用管同步，插件后台自动搞定！

## 四、手机端云同步设置（Android / iOS 通用）

### 1. 初始传输（只做一次）
- 用数据线把整个 vault 文件夹复制到手机
- 手机 Obsidian → Open folder as vault → 选中文件夹

### 2. 生成 GitHub Personal Access Token
- GitHub → Settings → Developer settings → Personal access tokens (classic)
- Generate new token (classic)
  - Expiration：No expiration
  - Scope：只勾选 **repo**
- **复制 Token（只显示一次！）**

### 3. 手机端配置 Git 插件
- Author name：GitHub 用户名
- Author email：注册 GitHub 时用的邮箱
- Personal Access Token：粘贴刚才的 Token
- 其他设置和电脑端一致

## 五、核心注意事项（血泪经验）

| 问题 | 解决方案 |
|------|----------|
| 冲突 | 不要同时在手机和电脑编辑同一个文件 |
| 冲突恢复 | Git 会生成 `.md.orig` 文件，手动对比修改后重新 Commit |
| 版本回退 | GitHub Desktop → Discard changes 或查看历史版本 |
| 中文编码 | 可加 `.gitattributes` 文件处理 |
